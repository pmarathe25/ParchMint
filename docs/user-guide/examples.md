# Examples

> Read when choosing a prepared project for learning or manual verification.

Open an example directory with **Open project**. Local `.parchmint/` state may
be created and can be deleted safely.

| Example | Focus |
|---|---|
| [`tour`](../../examples/tour) | Small guided overview |
| [`harbor-lights`](../../examples/harbor-lights) | First edit and export |
| [`medium-novel`](../../examples/medium-novel) | Binder depth and compile order |
| [`research-heavy`](../../examples/research-heavy) | Research and split panes |
| [`unicode-notes`](../../examples/unicode-notes) | Unicode, bidirectional text, counts |

Malformed, migration, recovery, edge, and scale inputs belong under
`tests/fixtures/`; they are not user projects.
