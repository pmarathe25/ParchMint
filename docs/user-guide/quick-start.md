# Quick start

> Read when creating and navigating a first project.

1. Launch ParchMint and choose **New project**.
2. Name the project and choose its parent folder.
3. Select **Manuscript**, then create a scene from **Structure**.
4. Add a title and synopsis in the inspector.
5. Write ParchMint Markdown; `Ctrl/Cmd+S` forces a save flush.
6. Add a research note and open the split workspace when you need reference material beside the draft.
7. Use `Ctrl/Cmd+1`, `2`, or `3` for Editor, Outline, or Cards in the focused pane.
8. Search from the top bar; use `Ctrl/Cmd+F` inside the current document.
9. Choose **Project → Export**, select a preset and format, then choose a destination.

Structural undo is separate from editor text undo. Deleting a binder item moves
it to project trash instead of immediately erasing its canonical files.

For prepared content, open [`examples/tour`](../../examples/tour).
