# Binder and summaries

The left binder shows manuscript and research roots. Groups and documents may
be nested arbitrarily. Select a node to edit its title, synopsis, labels, and
status in the inspector; synopsis planning does not require opening body text.

Move a node with drag/drop or the move/indent/outdent shortcuts. Delete moves a
subtree to recoverable project trash. Undo and redo structural commands operate
in Rust, so reordering does not alter the text editor undo stack. Compile order
is deterministic binder preorder, with manuscript before research.
