# Search and statistics

Project search covers titles, synopses, manuscript bodies, and research. The
SQLite index is disposable and can be rebuilt from canonical files; an index
failure never makes the project authoritative data unavailable. Search results
carry the document identity and counts follow the same Unicode word/character
rules as compile statistics.

Use the scope controls to limit results to a subtree or research. Rebuilding a
corrupt or incompatible cache is safe because source files are not rewritten.
