# parchmint-compile

> Read when changing binder traversal, compile IR, validation, cancellation, or exporters.

Consumes immutable snapshots and prepares deterministic Markdown, text, HTML,
PDF-render, EPUB, and DOCX output. It never reaches into live workspace state.
Format promises live in [Export fidelity](../../docs/reference/export-fidelity.md).

Check: `cargo test -p parchmint-compile`.
