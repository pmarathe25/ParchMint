# parchmint-markdown

> Read when changing front matter, Markdown parsing, semantic AST, diagnostics,
> opaque preservation, or deterministic serialization.

Owns the source-aware ParchMint Markdown codec. Does not own project storage or
Qt documents. Compatibility behavior is normative in
[ParchMint Markdown 1](../../docs/format/parchmint-markdown-1.md); fixtures live
under `tests/fixtures/markdown/`.

Check: `cargo test -p parchmint-markdown`.
